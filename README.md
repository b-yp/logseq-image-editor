- # loseq-image-editor
  - Edit and save images in Logseq.
- ## Features
  - For details: https://ui.toast.com/tui-image-editor#features
- ## Shortcut keys
  - `ESC`
    - Close the editor
  - `Ctrl/Command + Enter`
    - Save the image and close the editor
